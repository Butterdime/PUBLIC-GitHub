# 📊 Perplexity Dashboard

A modern, feature-rich dashboard built with Streamlit that provides comprehensive item management with a clean, professional interface.

## ✨ Features

### Core Functionality
- **➕ Add Items**: Create items with name, value, description, and category
- **🔍 Search & Filter**: Real-time search by name/description and category filtering
- **✏️ Edit Items**: In-place editing with form validation
- **🗑️ Delete Items**: Safe deletion with confirmation
- **📊 Statistics**: Category breakdown and item counts

### Data Management
- **📥 Export Data**: Download all data as JSON with timestamps
- **📤 Import Data**: Upload and merge JSON files
- **🔗 API Simulation**: View data as REST API responses
- **💾 Session Persistence**: Data maintained during session

### User Interface
- **📱 Responsive Design**: Works on desktop and mobile
- **🎨 Modern Styling**: Clean, professional appearance with custom CSS
- **📋 Organized Layout**: Sidebar controls, main content area, and management panel
- **🏷️ Categorization**: Organize items by General, Important, Archive, or Draft

## 🚀 Quick Deploy to Streamlit Cloud

1. **Fork/Clone**: Copy the files to your GitHub repository
2. **Connect**: Link your repo to [Streamlit Community Cloud](https://share.streamlit.io)
3. **Deploy**: Your dashboard will be live with a shareable URL
